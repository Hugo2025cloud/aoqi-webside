# 快速部署指南

## 当前仓库状态
- ✅ Git仓库已配置
- ✅ 工作目录干净
- ❌ 尚未添加远程GitHub仓库

## 步骤1：在GitHub上创建仓库

1. 打开浏览器，访问 [GitHub](https://github.com/)
2. 登录您的GitHub账号
3. 点击右上角 "+" 图标 → "New repository"
4. 输入仓库名称：`aoqi-webside`
5. 选择公开（Public）仓库
6. **重要**：不要勾选 "Initialize with README"
7. 点击 "Create repository"

## 步骤2：添加远程仓库

在PowerShell中执行：
```powershell
# 替换 yourusername 为您的GitHub用户名
$env:Path += ";C:\Program Files\Git\bin" 
git remote add origin https://github.com/yourusername/aoqi-webside.git
```

## 步骤3：验证远程仓库

执行：
```powershell
$env:Path += ";C:\Program Files\Git\bin" 
git remote -v
```

预期输出：
```
origin  https://github.com/yourusername/aoqi-webside.git (fetch)
origin  https://github.com/yourusername/aoqi-webside.git (push)
```

## 步骤4：推送代码

执行：
```powershell
$env:Path += ";C:\Program Files\Git\bin" 
git push -u origin master
```

## 步骤5：启用GitHub Pages

1. 在GitHub仓库页面，点击 "Settings"
2. 选择 "Pages" 选项
3. 在 "Source" 中选择 "Branch: master"
4. 选择文件夹："/root"
5. 点击 "Save"

## 访问您的网站
- 网站地址：`https://yourusername.github.io/aoqi-webside/`
- 可能需要几分钟时间生效

## 如果遇到问题

1. **网络问题**：检查网络连接或尝试使用VPN
2. **权限问题**：确保您的GitHub账号有权限推送代码
3. **命令错误**：检查命令中的用户名和仓库名称是否正确

如需更多帮助，请查看 `部署故障排除.md` 文件。
